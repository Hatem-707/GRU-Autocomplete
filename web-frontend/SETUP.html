üöÄ Quick Start (3 Steps)

Step 1: Generate ONNX Model

Run your Jupyter notebook save_model.ipynb to create:

  • model_daily_gru_attention.onnx (~90 MB)
  • word2id_daily.json vocabulary
  • id2word_daily.json vocabulary

These files should be in the autocomplete/ directory.

Step 2: Prepare Web Assets

Run the preparation script to create vocabulary and copy the model:

cd autocomplete/web-frontend
python3 prepare_assets.py

‚ùì Troubleshooting

Problem: Model fails to load

Solution:

  • Verify prepare_assets.py ran successfully
  • Check that model.onnx and vocabulary.json exist
  • Open DevTools (F12) ‚Üí Console for error messages
  • Check Network tab to see if files are loading

Problem: Predictions are slow

Solution:

  • First inference is slower (WebAssembly JIT compilation)
  • Subsequent predictions should be much faster
  • Try closing other browser tabs to free memory
  • Chrome generally has better WebAssembly performance

Problem: Port 8000 already in use

Solution:

Edit the port number in server.py:

PORT = 8001  # Change to any unused port
